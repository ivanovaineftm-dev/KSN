# Excel Processor (FastAPI)

Веб-приложение для загрузки, обработки и выгрузки Excel-файлов (`.xlsx`, `.xls`).

## Функционал

- Загрузка трех файлов через веб-форму:
  - основной файл с данными сотрудников;
  - справочник "Локации";
  - файл "Должность" (справочник "Бариста").
- Поддержка форматов `.xlsx` и `.xls` (Microsoft Excel 97-2003).
- Сохранение исходных файлов на сервере в папке `uploads/`.
- Обработка файла по правилам:
  1. Оставляются только строки, где столбец **C** содержит слово `стажер`.
  2. Удаляются строки, где столбец **H** пустой.
  3. Удаляются строки, где столбец **H** содержит февраль (включая даты вида `xx.02.2026`).
  4. Строка выделяется бледно-красным, если столбец **F** (должность наставника) пустой.
  5. Строка выделяется бледно-красным, если должность наставника в **F** не входит в допустимый список для должности стажера в **C**.
  6. Поддерживается регистронезависимая проверка и нормализация вариантов с тире/пробелами (`Повар`, `повар`, `кассир - стажер`, `кассир-стажер` и т.д.).
  7. Все строки с ошибками валидации подсвечиваются бледно-красным (`#FFC7CE`).
  8. Добавляется столбец `Подразделение (Hr-index)`.
  9. Для сопоставления подразделений используется столбец **D** из основного файла и столбец **B** из файла "Локации":
     - сравнение регистронезависимое;
     - лишние пробелы игнорируются;
     - используется только полное совпадение;
     - при отсутствии совпадения записывается `Не найдено`.
  10. Дополнительная замена подразделения выполняется по файлу "Должность":
      - только если в столбце **F** основного файла указано `бариста`;
      - значение из столбца **E** (наставник) ищется в столбце **C** файла "Должность";
      - поиск регистронезависимый с нормализацией пробелов, поддерживает точное/частичное/fuzzy совпадение;
      - при совпадении значение из столбца **B** файла "Должность" подставляется в столбец **D** основного файла.

Допустимые пары `Должность` (C) → `Должность наставника` (F):
- `повар-стажер` → `повар`, `повар-универсал`
- `повар-универсал стажер` → `повар`, `повар-универсал`, `кассир`, `старший кассир`
- `кассир - стажер` → `кассир`, `старший кассир`, `повар-универсал`
- `бариста - стажер` → `бариста`
- `старший кассир - стажер` → `старший кассир`, `заместитель директора`
- `работник торгового зала - стажер` → `старший кассир`, `кассир`, `работник торгового зала`, `повар-универсал`

- Сохранение обработанного файла в папке `processed/`.
- Автоматическая выгрузка (скачивание) обработанного файла пользователю.

## Технологии

- Backend: **FastAPI**
- Frontend: **HTML + JavaScript**
- Обработка Excel: **openpyxl + xlrd + pandas**

## Локальный запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте в браузере: `http://127.0.0.1:8000`
