# Excel Processor (FastAPI)

Веб-приложение для загрузки, обработки и выгрузки Excel-файлов (`.xlsx`, `.xls`).

## Функционал

- Загрузка двух файлов через веб-форму:
  1. Основной файл с данными сотрудников.
  2. Справочник «Локации» (опционально).
- Поддержка форматов `.xlsx` и `.xls` (Excel 97-2003).
- Валидация расширений загружаемых файлов.
- Обработка основного файла по правилам:
  1. Оставляются только строки, где столбец **C** содержит слово `стажер`.
  2. Удаляются строки, где столбец **H** пустой.
  3. Удаляются строки, где столбец **H** содержит февраль (включая даты вида `xx.02.2026`).
  4. Выполняется проверка соответствия должности наставника в **F** должности стажера в **C**.
- Сопоставление подразделений:
  - берётся значение из столбца **D** основного файла;
  - сравнивается со значениями столбца **B** файла «Локации» (без учёта регистра и лишних пробелов);
  - применяется exact, contains и fuzzy matching;
  - в результат записывается новый столбец **«Подразделение (нормализованное)»**.
- Если справочник не загружен или совпадение не найдено — записывается `Не найдено`.
- Сохранение результата в `processed/` в формате `.xlsx` без изменения исходных файлов.
- Автоматическая выгрузка обработанного файла пользователю.

## Технологии

- Backend: **FastAPI**
- Frontend: **HTML + JavaScript**
- Обработка Excel: **pandas + openpyxl + xlrd**

## Локальный запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте в браузере: `http://127.0.0.1:8000`
